Neighborhood Map

It is a simple app to keep track of destinations I'd like to visit. This project uses the JavaScript to show map data.We are using Google Map API to create markers,infowindow etc. It also makes AJAX call to get the wikipedia related links for the places when user clicks on any marker in the map. It uses the power of KnockoutJS MVVM framework. The location data is simple JSON object with predefined places.

Seesaw Blog

Seesaw is a small blog project with user loging and signup option.You need to login to create new blog post. Once you login, you will see a link to add post. The project is implemented using Python backend and uses Google Clout Platform as it hosting provider. Google App Engine's Data Store is used as storage.The DB results are cached and also the user login details are hashed. It uses cookies to authenticate user.
